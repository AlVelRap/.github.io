Tema 2 - Uso de estilos

Un estilo es una combinación predefinida de un tipo de fuente, color, tamaño de letra y distribución de elementos que puedes aplicar en cualquier página web. Aplicar un estilo puede ayudarte a mejorar el diseño y la presentación de tu página web.

En este tema veremos distintas formas de dotar de estilo a nuestra página web. Para ello recurriremos a distintos frameworks, como los que podemos ver aquí debajo. Tambien sentaremos las bases del uso de Flexbox.

Bootstrapmore_vert Link
Bootstrapclose

Framework de frontEnd potente, ampliable y repleto de funciones. Crea y personaliza con Sass, utilice el sistema Grid y los componentes prediseñados y haga que los proyectos cobren vida con potentes complementos de JavaScript.

Materializemore_vert Link
Materializeclose

Framework de frontEnd basado en Material Design. Enfocado en la experiencia del usuario, busca acelerar el desarrollo de aplicacioones Web y facilitar la forma en que se trabaja a traves de su documentación.

SASSmore_vert Link
SASSclose

SASS extiende al lenguaje CSS dotandole de nuevas caracteristcias como la posibilidad de crear variables, funciones, etc.
